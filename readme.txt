VoIPDown is an operation that aims to bring together as many Moroccan sites to protest against the blocking of VoIP.

== Description ==

This open source community project was created in response to the blocking of VoIP technologies in Morocco by the National Agency for Regulation of Telecommunications, since January 5, 2016.

To contribute, simply create a ticket on the project page on Github, or send an email to allo [AT] voipdown [DOT] com

== Installation ==



1. Download this plugin
2. Go to Wordpress Dashboard --> Plugins --> Add New --> Upload Plugin ( You can also upload via FTP)
3. Select the plugin file.
4. Install it.


== Frequently Asked Questions ==

No, once the plugin is activated, you'll see the badge on your website.
